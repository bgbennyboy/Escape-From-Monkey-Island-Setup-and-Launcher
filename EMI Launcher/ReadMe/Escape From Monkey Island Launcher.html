What is Escape From Monkey Island Launcher ?:
A replacement launcher program for Escape From Monkey Island (Monkey Island 4).
The original launcher program doesn't work on modern versions of Windows.
It also has features not present in the original launcher:

  • Can run EMI in windowed mode.

  • Can patch the game so the cds don't need to be in the drive to play.

  • Can make EMI run in a higher resolution (subject to some limitations, see the bugs/limitation section below).

  • Supports all recent versions of Windows (XP, Vista, Windows7, Windows8). Windows 98, ME and 2000 are not officially supported but should work correctly.

  • Can automatically download and install the V1.1 update patch for all language versions of EMI.

  • Loads one of 10 random images as the main launcher image - for added eye candy.

The program does not need to be installed and can be run from anywhere since it reads the registry values in order to find the location of EMI. So Escape From Monkey Island must be installed for the launcher to work correctly.


How to use it:
Usage should be fairly self-explanatory:

  • Click either Play fullscreen or Play windowed to start the game.

  • The Lucasarts readme button displays Lucasarts' EMI readme, while the Launcher readme button displays this readme.

  • Click Change Resolution to bring up the Change Resolution window. Choose a resolution from the dropdown box and then click 'Change' to change the resolution that EMI runs in.

    The resolution patch has been tested on patched and unpatched English versions of EMI. If you use a version of EMI for another language it *should* still work but this is untested.

Click 'Options' to bring up the options menu.

  • Run without cds:
    This patches the game so that the cds dont have to be in the drive to play. If you click it again, then it will undo the patch. For this to work - you must have installed the patch to update the game to version 1.1. It should work with all versions of the game except Italian.

  • Get update patch:
    This automatically downloads and installs the V1.1 game update.

  • Close launcher on play:
    If enabled, then the launcher closes when one of the play buttons is clicked.

  • Direct3D/OpenGL:
    Lets you choose between the two rendering modes.

  • 32 bit/16 bit colour:
    Allows you to switch between the different colour modes.

  • Sounds in launcher:
    If enabled, then the launcher plays the same sounds that Lucasarts' original launcher program did.


Bugs/Limitations:
The no cd patch doesn't work with the Italian version. For some reason this version of the game has different copy protection to the others.

There are known issues with the resolution patch.:

  • Cutscenes may not fill the whole screen. The patch makes the videos double in size at resolutions of 1280x960 and above. But if you choose a resolution other than 640x480 or 1280x960 the videos will not fill the whole screen.

  • Playing in a window wont work with some resolutions. Fullscreen will still work though.

  • There may be some visual glitches like tearing.

  • Saved games made in higher resolutions are not always compatible with each other. They will usually work but there may be visual glitches if you load a saved game that you made in a different resolution.
